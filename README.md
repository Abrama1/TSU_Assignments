# HW_3

## How to run
1) Generate input files:
   - Run `eq_gen.py` to create `file_*.in` files.

2) Process files:
   - Run `python main.py`

## Output
- The program prints the name of the `.in` file whose solution vector has the maximum norm.
- Each input file produces a corresponding output file: `file_k.in.out` containing the computed norm.
